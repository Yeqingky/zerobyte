## Zerobyte 是什么？

Zerobyte 通过在 Restic 强大的备份引擎之上提供直观的 Web 界面，简化了备份管理。无论你是备份本地目录、NFS 共享还是云存储，Zerobyte 都会自动处理加密、压缩、调度和保留策略。

### 关键能力

- **自动化备份**，支持加密、压缩和保留策略
- **灵活的调度**，支持类 cron 表达式
- **多协议支持**，支持 NFS、SMB、WebDAV、SFTP 和本地目录
- **云存储**，支持 S3、Google Cloud Storage、Azure Blob 及 40 多种 rclone 后端
- **快照浏览与恢复**，直接在 Web 界面中操作

<Callout type="warn">
  Zerobyte 仍处于 0.x.x 版本，各版本之间可能会有重大变化。核心功能正在积极开发中
</Callout>
